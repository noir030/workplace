# Run <picture> <source srcset="https://fonts.gstatic.com/s/e/notoemoji/latest/1fae5/512.webp" type="image/webp"> <img src="https://fonts.gstatic.com/s/e/notoemoji/latest/1fae5/512.gif" alt="ðŸ«¥" width="32" height="32"> </picture>
### 1. Clone the repository and navigate to the project directory:
```
git clone https://github.com/noir030/workplace.git
cd workplace
```
### 2. Create a directory for the build and go to it:
```
mkdir build
cd build
```
### 3. Compile the project with CMake:
```
cmake ..
make
```
### 4. Run the program:
```
./workplace
```
